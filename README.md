# APKPAC Frontend

## Features
- React.js (Vite)
- Tailwind CSS
- React Router DOM
- React Quill (admin rich text)
- Modern, responsive UI
- Admin panel for app/blog/topic management
- Public pages: Apps, Games, Topics, Blogs, Tools, Search, Footer pages
- SEO-friendly routes

## Scripts
- `npm run dev` — Start dev server
- `npm run build` — Build for production
- `npm run preview` — Preview production build

## .env Example
```
VITE_API_URL=http://localhost:5000/api
```
